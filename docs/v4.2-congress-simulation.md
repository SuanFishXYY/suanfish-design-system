## TC 总结 · v4.2 协议的关键演示

1. **4:4:4 均权达成**: 5 Tier 0 入场 = 3 哲 + 1 艺 + 1 音, 强制三大类各 ≥1
2. **加减辩证内化**: [+] 派 (达芬奇/贝多芬/莫奈) 与 [-] 派 (王弼/凯奇/倪瓒) 在议会内辩论, 不是 LLM 和稀泥
3. **禁一票否决 (P1-6)**: 凯奇/倪瓒虽倾向 reject 动效/加法, 仍通过表决支持 — 反对意见转化为约束
4. **跨类邀请 (Step 3)**: 达芬奇邀巴赫 / 贝多芬邀莫奈 — 跨类共振 ≠ 单独发言
5. **R-Cross1-4 全部命中**: 艺音 Tier 0 现在也有 R 规则锚 (P2-8 修)
6. **task_kind user-declared (P1-5)**: BRIEF 显式声明 mixed, 无 LLM 自利路由风险

> **结论**: v4.2 议会不是把"减法独裁"换成"加法独裁", 而是把"独裁"换成"辩证民主"。
